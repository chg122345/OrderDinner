#IHotel
* 技术
    * [x] 基于Servlet 3.1 标准改造的Controller
        * 类似于Spring MVC风格
    * [x] [Jleopard](http:www.jleopard.org) ORM框架
* 数据库
    * [x] MySQL数据库
* 项目详情
    ```
    -org.jleopard.ihotel
        -config  项目基础配置，数据库信息配置
        -controller Controller类
        -core 实现Spring MVC风格核心注解等
        -dao 数据持久层
        -entity 实体类
        -filter 过滤器
        -service 业务逻辑层
        -util 工具包
        -DispatcherServlet 最最最核心Servlet
    ```
